<objective>
Deliver the first vertical slice of cross-galaxy travel: extend the body model with explicit `Type` routing + galaxy presentation fields, author 3 real flyable galaxies at true 1:1 intergalactic distances, and render them procedurally in the sky shader so they grow from sub-pixel points into spiral/elliptical discs on approach.

Purpose: This is the headline visual of the phase (Elite/Frontier procedural galaxies) AND the data + routing foundation every later slice consumes (TRV-02 galaxies, RND-05 galaxy-only-sky, RND-07 handoff). It also de-risks the one open shader-uniform question (Godot `int[]` uniform arrays) before any dependent slice is built.

Output: A running game where, from the home system, the player sees the 2 other galaxies as distinct colored procedural discs fixed to world space; the universe now contains 3 galaxies with correct SOI and physical radii ready for intergalactic flight (Plan 03-02) and Galaxy-space mesh promotion (Plan 03-03).
</objective>
