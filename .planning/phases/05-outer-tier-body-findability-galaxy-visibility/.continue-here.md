# BLOCKING CONSTRAINTS — Read Before Anything Else

> These are not suggestions. Each constraint below was discovered through failure.
> Acknowledge each one explicitly before proceeding.

- [ ] CONSTRAINT: Do NOT re-attempt the godot-starlight "manual clip-space `POSITION` + far-plane-clip" MultiMesh billboard technique for stars. It was tried exhaustively in 05-01 and abandoned. Any re-plan must choose a structurally different, robust rendering approach.

**Do not proceed until all boxes are checked.**

## Critical Anti-Patterns

| Pattern | Description | Severity | Prevention Mechanism |
|---------|-------------|----------|---------------------|
| Manual clip-space billboard MultiMesh for stars (godot-starlight port) | A `StarPointRenderer` MultiMesh wrote `POSITION` directly in the vertex shader with a far-plane-clip trick. It chained ~8 separate engine-specific failures before rendering at all: script not attached (ext_resource missing `uid`), `_Ready` NRE (child `_Ready` runs before parent `TestSetup` populates `GameObjects`), back-face culling of billboard winding, NDC-z = -w outside Vulkan [0,1] clip range, `ALBEDO` flooding the quad in unshaded mode, `mix(min,1,bright)` scaling quads to full-screen, and frustum-culling of the whole `MultiMeshInstance3D` (no custom AABB). Even after all fixes it still did not render usably. | blocking | On re-plan, pick a robust path instead: either extend the existing **skybox** shader to also paint the CURRENT tier's stars (not just next-tier-out), or use standard Godot 3D billboards/sprites at a FIXED in-frustum distance along each star's true direction. Never hand-write clip-space POSITION + far-plane-clip again. |
| Unbounded fractal overlay in a sky shader | The original D-54 "Star-Nest" overlay used `abs(p)/dot(p,p)-c` with a `pow(.,-2)` singularity over the whole hemisphere; it clamped to 1 everywhere and painted the sky brown. | advisory | Keep volumetric grain BOUNDED and MULTIPLIED into an already-localized disc brightness (as the kept `skybox.gdshader` now does), so it can never fill the sky. |
