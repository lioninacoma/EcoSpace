## Trust Boundaries

Single-player Godot game — no network, auth, sessions, or PII. The realistic threat surface
is invalid-state / numeric-stability in the draw path, not classic security.

| Boundary | Description |
|----------|-------------|
| `WorldRenderer.GetRenderPosition` → Hud draw | Render-space position read each frame; a missing/stale entry must not crash or mis-place the circle. |
| `Camera3D.UnprojectPosition` → screen coords | Behind-camera positions wrap to the opposite side of the viewport (screen-space flip). |
| Inspector circle radius knobs → draw | MIN_R/MAX_R clamp bounds the drawn radius. |
