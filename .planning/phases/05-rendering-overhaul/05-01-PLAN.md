<objective>
Build the foundation of the unified luminous-body renderer: the per-body `LuminousBodyDescriptor` struct, the pure `LuminousLod` distance→LOD-weight curves, and the `LuminousDescriptorBuilder` that runs ONE classify→project→appearance loop per frame (migrated from `SkyboxRenderer.SyncSkyPoints`). The existing skybox stays running and visually unchanged — this plan adds the new data pipeline alongside it and validates projection + precision with zero visual regression.

Purpose: Establish D-02's single source of truth (one descriptor, shared by all drawers) before any drawing changes, so Plans 2–4 can route the same numbers to different drawers and pops become mathematically impossible.
Output: `LuminousBodyDescriptor`, `LuminousLod`, `LuminousDescriptorBuilder` + new unit tests; builder node wired into Main.tscn at deterministic process priority. No visual change.
</objective>
